Щоб запустити цей проєкт, виконайте наступні дії:

1. Переконайтеся, що у Вас встановлено Node.js та середовище розробки (наприклад, Visual Studio або Visual Studio Code), яке підтримує React.
2. Завантажте вміст цього репозиторію на комп'ютер, написнувши Code -> Download ZIP.
3. Розпакуйте ZIP-архів у зручну папку.
4. Відкрийте папку у середовищі розробки, натиснувши "Відкрити локальну папку".
5. Відкрийте термінал (Вид -> Термінал -> Командний рядок розробника).
6. Переконайтеся, що шлях до папки проєкту виводиться у терміналі коректно. Інакше - введіть інший шлях за допомогою команди "cd *Шлях*".
7. Виконайте команду "npm run dev".
8. Перейдіть за адресою, вказаною у терміналі.
